Архитектор собирается спроектировать новое самое высокое здание в городе.
Для этого ему сперва необходимо определить, какое из существующих зданий является самым высоким.
Помогите архитектору найти самое высокое здание в городе.

Входные данные: Высота зданий как двумерный {% if is_js %}массив{% else %}список{% endif %}. Высота здания определяется как столбец в {% if is_js %}массиве{% else %}списке{% endif %}.

Выходные данные: Номер дома и его высота как {% if is_js %}массив{% else %}список{% endif %} целых чисел (Примечание: в этой задаче, как и в реальной жизни, номера домов начинаются с "1", а не с "0")

  • array height <= 10;
